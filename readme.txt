== Description ==

Enables users to import Notes from Evernote Account.

This plugin use WP-Cron to import and update Notes from Evernote.

Of course, users can import notes from their own accounts.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-evernote-synchronizer` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= how to set up? =

Go to "Evernotes" -> "Settings".

For Individual settings, Go to "Users" -> "Your Profile".
Other Users including Admin Cannot See other's token.
